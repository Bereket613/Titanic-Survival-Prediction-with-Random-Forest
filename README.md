# Titanic-Survival-Prediction-with-Random-Forest
This project uses the Titanic dataset to predict passenger survival using a machine learning pipeline. It includes feature engineering, data preprocessing, and model optimization with Random Forest and GridSearchCV.

## 🚀 Features
- Feature engineering: Family size, titles, cabin info, age/fare bins
- Preprocessing pipelines: Imputation, scaling, encoding
- Model training: Random Forest Classifier
- Hyperparameter tuning: GridSearchCV
- Feature selection: SelectFromModel
- Final predictions saved for Kaggle submission
